# FullStackWebApplication
Repository for Full Stack Web application in GO

## Frameworks used:

- Gin - GO framework
- GORM - ORM in GO
- React for frontend
- Postgres for DB
- OAUTH2*
- JWT*

### Why Gin instead of Mux ?

- https://golang.company/blog/comparison-between-gin-gorilla-mux-and-net-http (Gin is easy to understand)
